## Installation

### With CMake

```bash
git clone https://github.com/terrakuh/fast_cgi.git
cd fast_cgi

mkdir build
cd build

cmake ..

# cmake -DFAST_CGI_BUILD_EXAMPLES=OFF

cmake --build .
cmake --build . --target install
```

### Without CMake

Just copy the *fast_cgi* folder with *.hpp* files to your project or set it as include directory.
